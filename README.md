# Four Card Feature Section

A responsive **Four Card Feature Section** built as part of the *Frontend Mentor* challenge.  
This project focuses on constructing a feature section layout with multiple cards using HTML and CSS, following a professional design specification. 

🛠 Features
Responsive card layout
Flexbox or CSS Grid for alignment
Clean typography, spacing, and visual hierarchy
Uses semantic HTML

📈 What I Learned
Example points you can customize:
Built complex layout using only HTML & CSS
Practiced responsive design with media queries
Improved use of semantic elements and CSS structure

## 📌 Overview

This repository contains a feature section with four cards:

- Each card highlights a feature with an icon, title, and description  
- Clean, minimal design with consistent spacing and typography  
- Fully responsive layout that adapts to different screen sizes  

This project was built to practice layout techniques such as Flexbox/Grid while mirroring a provided design file. :contentReference[oaicite:2]{index=2}
